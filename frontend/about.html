NexusNotes Dashboard

ABOUT

Built to organize student workflows end-to-end.

NexusNotes is a student-focused frontend for discovering subjects, opening curated resources, and keeping study activity visible in one dashboard. The project uses a multi-page Vite setup, so each page can be directly opened with a clean URL.

Core flow: landing page at localhost:5173, login/signup for access, and a dashboard that opens subject-level pages for notes, question banks, and projects.
